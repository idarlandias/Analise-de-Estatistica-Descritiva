# Análise de Estatística Descritiva

Este repositório contém um notebook em Python que realiza uma análise estatística descritiva em uma amostra do estudo "Americans' Changing Lives".

## Análises Realizadas
- Verificação da distribuição de idade, peso e altura.
- Cálculo de medidas de tendência central (média, mediana).
- Análise de dispersão com o Coeficiente de Variação (CV).
- Verificação da correlação entre peso e altura.

## Ferramentas Utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
